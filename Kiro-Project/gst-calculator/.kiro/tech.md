# Technology Stack

## Core Technologies
- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern styling with gradients, flexbox, and transitions
- **Vanilla JavaScript**: No frameworks or build tools required

## Build System
None required - this is a static web application.

## Running the Application
```bash
# Simply open the HTML file in any modern browser
start index.html  # Windows
open index.html   # macOS
xdg-open index.html  # Linux
```

Or use a local development server:
```bash
# Python 3
python -m http.server 8000

# Node.js (if http-server is installed)
npx http-server
```

## Browser Compatibility
Targets modern browsers with ES6+ support. No transpilation needed.

## Deployment
Static hosting compatible - can be deployed to:
- GitHub Pages
- Vercel
- Netlify
- Any static file server
